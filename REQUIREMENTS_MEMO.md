# 録音録画・配信・YouTubeアップロード ワンクリックアプリ 要望メモ

作成日: 2026-04-20

---

## やりたいこと（一言）

礼拝の録画・Zoom配信・YouTube投稿を、ボタン1つで完結させたい。

> **背景メモ（2026-04-20）:**
> 今回は人員不足のため音声のみ収録・手動変換でYouTubeにUPした。
> 本来はカメラ映像あり。映像録画は必須要件。

---

## ユーザーストーリー

1. アプリを起動する
2. 礼拝情報（日付・タイトル・聖書箇所・説教者）を入力する
3. 「開始」ボタンを押す → 録画・Zoom配信が自動スタート
4. 礼拝終了後「終了」ボタンを押す → 録画停止・サムネイル自動生成・YouTubeアップロード

---

## 必要な機能

### 1. 録画・音声収録
- **映像録画は必須**（カメラ映像 + マイク／ライン音声）
- 今回（2026-04-20）は人員不足で音声のみだったが、通常は映像あり
- 形式: MP4（YouTube直接アップロード可能）
- 選択肢A: OBSをバックエンドに使う（OBS WebSocket API経由で制御）
  - カメラ切替・テロップ合成など多機能
- 選択肢B: 独自ツール（FFmpeg ベース）で録画
  - メリット: OBSインストール不要、軽量、自動化しやすい
  - デメリット: 映像合成・ミキサー機能はOBSより劣る
- → **OBS連携が現実的**（すでに礼拝で使用しているなら特に）

### 2. Zoom配信
- Zoom ミーティングの自動開始（Zoom SDK or Zoom OAuth API）
- または「Zoom カスタムライブストリーミング」機能でRTMPストリームを受け取る
  - OBS / FFmpegからZoom RTMPエンドポイントに配信

### 3. サムネイル自動生成
- 礼拝情報入力から thumbnail.png を自動生成（Pillow）
- テンプレート: 調布南キリスト教会デザイン（今回作成済み）

### 4. YouTube自動アップロード
- YouTube Data API v3 を使用
- タイトル・説明文・サムネイルを自動セット
- タイトル例: `2026年4月19日 「忘れ得じ、彼のパリサイびと」 ヨハネの福音書3章1〜15節 前田重雄師`
- 公開設定: 限定公開 or 公開（選択式）
